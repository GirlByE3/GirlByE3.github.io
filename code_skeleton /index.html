Let's look at some forms!

Text-related Form Elements

Text Input

Each input should a unique name attribute

What is your name?

Textarea Input

Each input should a unique name attribute

Each Textarea should have unique 'name' attribute

Option-related Form Elements

All checked inputs should a unique (but similar) "name" attribute and different 'value' attribute

How do get to school?

Radio Input

All radio inputs of the same group should have the Same 'name' attribute, but differnet 'value' attribute

What program are you in?

Complex Form Elements

Date Input

Each date input should have a unique 'name' attribute


Dropdown (select/options)

Each dropdown should a unique 'name'attribute on the 'select' element, and unique 'value' attribute on each 'option' element.

Button-related Form Elements

Button Inputs

A button input doesn't need a 'name' attribute, but to change the text inside the button, you will need to use the 'value' attribute.
